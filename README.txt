Lift Personalization for Wordpress.

== Description ==

Lift Personalization for Wordpress. Does not support syndication. Creates a simple personalization custom post type to create content.

== Installation ==

1. Download this repository
2. Run composer update
3. Place plugin in wp-content/plugins directory
4. Activate
5. Fill out credentials at Settings > Quicklift
